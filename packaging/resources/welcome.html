Claude Mate

This installs the Mac side of Claude Mate — the daemon that bridges Claude Code session status to your USB hardware status device (OLED + traffic light).

The installer will:

  • Install the daemon to /Library/Application Support/ClaudeMate
  • Install the status hook into ~/.claude/hooks/
  • Merge the Claude Mate hooks into your ~/.claude/settings.json (your existing settings are preserved)
  • Start the daemon now and at every login (LaunchAgent)

Flashing the Arduino firmware is a separate step — see the project docs/INSTALL.md and docs/WIRING.md.
